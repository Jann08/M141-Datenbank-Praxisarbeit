## Ausgangslage

Eine Jugendherberge (Backpacker) betreibt ihre Datenbank in Microsoft Access. Diese soll auf MariaDB/MySQL migriert werden. Die bestehende Struktur und Daten liegen als DDL-Skript und CSV-Datei vor.

---

## Projektziel (SMART)

**Spezifisch**: Migration der Datenbank `backpacker_lb3` von Access auf MariaDB lokal und auf ein gesichertes Cloud-DBMS.

**Messbar**: Alle Daten vollständig importiert, FK-Constraints gesetzt, Berechtigungen gemäss Zugriffsmatrix, Migrationstests bestanden.

**Akzeptiert**: Abnahme durch Lehrperson mit Demo (10-15 Min., 3 User auf Cloud-DBMS).

**Realistisch**: Umsetzbar in 9-12 Lektionen + Heimarbeit.

**Terminiert**: Abgabe gemäss Meilensteinplan (Tag 8/9/10).
